## Contributing

Thanks for contributing to OpenCode Provider Status.

### Before opening a pull request

- Open an issue describing the improvement or bug.
- Keep changes focused and easy to review.
- Test your change against a live Ollama and Copilot endpoint.

### Local setup

```bash
npm install
npm run build
npm start
```

### Pull request checklist

- Explain what changed and why.
- Link the related issue.
- Note any new provider dependencies.
